# Planks

## Abbreviations

__E.S__ - Euclidean Space

## Prerequisites

### Symetric Convex Body

#### Compact Set

__Definition__ - A subset of a _E.S_ is called _compact_ iff it is _closed_ and _bounded_.

> The general definition of a compact set is more abstract, but equivalent in a __E.S__ <sup>[1](https://en.wikipedia.org/wiki/Heine%E2%80%93Borel_theorem)</sup>

__Examples__

- the set { x such that norm(x) < 1 } is open, therefor not compact.
- the set of upper half plane is unbounded, therefore note compact
_ the set { x such that norm(x) <= 1 } is closed and bounded, therefore compact

#### Convex Set

__Definition__ - A subset of a _E.S_ is called _convex_ iff it contains the line segment between every two points in it.

- the segment (-1, 1) is convex
- the segment (-2,-1)V(1,2) is not convex
- a donut is not convex
- a ball is convex

#### Convex Body

__Definition__ - A _compact_, _convex set_ that has a non-empty interior.

> the interior of a set is another topological definition; specifically, we will indicate that a set with non-empty interior is a set which has a point such that there exists a neighbourhood of it which is included in the subset.
